### 我理解的闭包：在作用域外部能够访问其作用域内部的变量，就构成闭包。

1. 举例说明
   ```JavaScript
    const f1 = () => {
      let n = 1;
      const f2 = () => {
        n+=1;
        console.log(n);
      }
      return f2;
    }
    const foo = f1();
    foo();//2
   ```
2. 闭包的优点与缺点
   - 优点：可以隐藏函数内部的实现细节，只暴露一个返回值供外部访问。
   - 缺点：在IE浏览器下，可能会导致内存泄漏。
